# Tổng hợp bài học Lập trình thi đấu

Bộ bài học **Lập trình thi đấu** được biên soạn dành cho người mới bắt đầu, từ các kiến thức nền tảng đến nâng cao. Mỗi bài học đều có giải thích trực quan, ví dụ minh hoạ và code mẫu. Hãy học theo thứ tự từ bài 1 để đạt hiệu quả tốt nhất!

- **[Codeforces](https://codeforces.com/)** - Các contest online, phân loại bài theo độ khó
- **[AtCoder](https://atcoder.jp/)** - Contest chất lượng cao, bài toán hay
